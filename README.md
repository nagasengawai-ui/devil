# CODIN - Engineering Scalable API & IT Solutions

Production-ready, enterprise-grade website for CODIN. PHP (modern MVC-like), HTML5, CSS3, JS, GSAP/AOS animations, responsive and SEO-optimized. Ready to deploy on PHP 8+ with Apache or Nginx.

Structure:
- public/ (web root)
- src/ (Controllers, Models, Views)
- config/
- storage/
- .env.example
- composer.json

See deploy.md for deployment instructions.
